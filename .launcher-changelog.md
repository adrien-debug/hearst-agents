# Changelog - Launcher & Renommage (23/04/2026)

## Modifications appliquées

### 1. Renommage du projet
- `hearst-agents` → `hearst-os` dans `package.json`

### 2. Scripts launcher créés
- `launch.sh` - Démarre tous les services (9000, 8100, 3000)
- `launch-stop.sh` - Arrête tous les services

**Fonctionnalités:**
- Kill automatique des processus existants
- Nettoyage des caches `.next`
- Démarrage séquentiel avec PIDs
- Vérification des ports après démarrage
- Logs dans `/tmp/hearst-*.log`
- Colorisation console (succès/erreurs)

### 3. Scripts npm ajoutés
```json
"launch": "./launch.sh",
"stop": "./launch-stop.sh"
```

### 4. Documentation
- `README.md` - Section Setup mise à jour avec ports et services
- `LAUNCHER.md` - Guide complet du launcher
- `.launcher-changelog.md` - Ce fichier

## Architecture des services

| Service | Port | Projet | URL |
|---------|------|--------|-----|
| hearst-os | 9000 | /Dev/hearst-os | http://localhost:9000 |
| hearst-connect | 8100 | /Dev/hearst-connect | http://localhost:8100 |
| Hearst-app | 3000 | /Dev/Hearst-app | http://localhost:3000 |

## Usage

```bash
# Démarrer tous les services
npm run launch

# Arrêter tous les services
npm run stop

# Voir les logs
tail -f /tmp/hearst-os.log
tail -f /tmp/hearst-connect.log
tail -f /tmp/hearst-app.log
```

- Les scripts sont exécutables (`chmod +x`)
