📖 Légende des Connexions

→ Flèche verte = appel direct
⇄ Flèche orange = synchronisation bidirectionnelle
··▶ Pointillés = validation ou permission
🔑 Icône jaune = authentification via Nango
🛡️ Icône rouge = validation sécurité
🧠 Icône grise = contexte utilisateur
🌐 Icône violette = connexion MCP

📌 Exemple Concret d'Exécution

🗣️ Demande utilisateur :
"Prépare un compte rendu de réunion, crée les tâches dans Linear, envoie le résumé par email et range le document dans Drive."
🎼
Orchestrateur comprend la demande multi-parties
🧠
Agent Mémoire récupère le contexte projet et préférences utilisateur
📄
Agent Bureautique crée le compte rendu formaté
📋
Agent Tâches crée les tickets Linear avec les bonnes priorités
🌐
MCP Router appelle les serveurs Google Drive, Gmail et Linear
🔐
Nango fournit les tokens OAuth pour accéder aux APIs
🛡️
Agent Sécurité demande confirmation avant l'envoi de l'email
✅
Orchestrateur assemble le résumé final et l'envoie à l'utilisateur
